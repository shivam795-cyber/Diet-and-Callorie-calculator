This project involves a simple calorie calculator built with Python. It helps figure out daily calorie needs for people. The tool relies on the Harris-Benedict formula to calculate basal metabolic rate. From there, it shows calories for maintaining weight, gaining some, or losing a bit.

üìç Features

The features include taking inputs from users. Those cover gender, age, weight, and height. It also factors in physical activity levels. Options range from sedentary lifestyles to light, moderate, or active ones. The calculator then works out the BMR. It gives daily maintenance calories too. Plus, it adjusts for goals like gaining or losing weight. Overall, this makes it easy for beginners. It draws from actual health formulas people use.

üßÆ Formula Used

The formula splits by gender. For males, BMR comes out as 88.36 plus 13.4 times weight in kilograms. Add 4.8 times height in centimeters. Then subtract 5.7 times age in years. For females, it starts at 447.6. Add 9.2 times weight in kilograms. Include 3.1 times height in centimeters. Subtract 4.3 times age in years. Activity levels multiply the BMR by specific factors. Sedentary uses 1.2. Light exercise goes with 1.375. Moderate exercise takes 1.55. Very active hits 1.725
